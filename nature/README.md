# 高中學測自然科互動式模擬考與學習診斷系統

## 🎯 系統簡介

這是一個完整的高中學測自然科模擬考試平台,提供68題符合學測規格的試題、互動式作答介面、自動批改功能,以及智慧學習診斷與補救建議。

## ✨ 主要功能

- ✅ **68題完整試卷** - 物理、化學、生物、地科各17題
- ✅ **互動式作答** - 題目導覽、進度追蹤、答案自動保存
- ✅ **自動批改** - 即時批改與成績計算
- ✅ **學習診斷** - 多維度分析學習盲點
- ✅ **補救建議** - 教師視角的具體教學建議
- ✅ **錯題檢討** - 詳細解析與常見錯誤提醒

## 🚀 快速開始

### 方法一:直接開啟(推薦用於測試)
```
直接用瀏覽器開啟 index.html
```

### 方法二:使用本地伺服器(推薦)
```bash
# 使用Python啟動簡易伺服器
python -m http.server 8000

# 然後在瀏覽器開啟
http://localhost:8000/index.html
```

### 方法三:使用VS Code Live Server
1. 安裝Live Server擴充套件
2. 右鍵點選index.html
3. 選擇「Open with Live Server」

## 📊 試題內容

### 科目分布
- **物理** - 17題(運動學、力學、電磁學、波動光學)
- **化學** - 17題(原子結構、化學鍵、反應、酸鹼、有機)
- **生物** - 17題(細胞、遺傳、演化、生態、生理)
- **地科** - 17題(地球結構、岩石、大氣、海洋、天文)

### 題目特色
- 符合近三年學測難度
- 素養導向命題
- 每題附詳細解析
- 標註關鍵概念與常見錯誤

## 💡 使用流程

1. **開始測驗** - 開啟index.html,系統自動載入試題
2. **作答** - 閱讀題目,選擇答案,系統自動保存
3. **交卷** - 完成後點選「交卷」按鈕
4. **查看成績** - 自動跳轉至成績報告頁面
5. **學習診斷** - 查看學習盲點分析與補救建議
6. **錯題檢討** - 仔細閱讀錯題解析

## 🎓 適用對象

## ⚠️ 注意事項

1. 建議使用Chrome、Edge或Firefox瀏覽器
2. 若遇到JSON載入問題,請使用本地Web Server
3. 作答過程中請勿關閉瀏覽器
4. 重新測驗前會清除之前的作答記錄
