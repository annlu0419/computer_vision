# 115學年度學測英文模擬試題 - 線上測驗系統

## 📚 系統簡介

這是一個完整的學測英文模擬試題線上測驗系統，專為高三考生最後衝刺階段設計。系統提供：

- ✅ 完整50題線上作答（詞彙、綜合測驗、文意選填、篇章結構、閱讀測驗、混合題）
- ✅ 即時計時（100分鐘）
- ✅ 自動評分（客觀題62分）
- ✅ 答案暫存（重新整理不會遺失）
- ✅ 詳細解析與學習回饋
- ✅ RWD響應式設計（支援手機/平板/電腦）

## 🚀 使用方式

### 開始測驗
1. 開啟 `index.html` 檔案
2. 系統自動開始計時（100分鐘）
3. 點選左側題號導覽可快速跳題
4. 選擇答案後自動儲存

### 作答介面
- **單選題**：點選選項即可作答
- **多選題**：可勾選多個選項（第49題）
- **填充題**：直接輸入答案（第47、48、50題）
- **翻譯/作文**：需另外作答（系統提供參考答案）

### 交卷與查看成績
1. 點選「交卷」按鈕
2. 系統自動計算客觀題分數（62分）
3. 顯示各大題得分情況
4. 翻譯（8分）與作文（20分）需教師或自我評量

### 查看詳解
- 開啟 `review.html` 查看完整答案解析
- 包含正確答案、解題重點、干擾選項分析

## 🎯 功能特色

### 1. 智慧題號導覽
- 50個題號按鈕，快速跳題
- 已作答題目顯示藍色
- 當前題目顯示綠色
- 即時顯示作答進度

### 2. 自動儲存機制
- 每30秒自動儲存作答狀態
- 使用 localStorage 技術
- 關閉瀏覽器後重新開啟可繼續作答
- 保留剩餘時間

### 3. 計時功能
- 100分鐘倒數計時
- 剩餘10分鐘時紅色閃爍提醒
- 時間到自動交卷

### 4. 自動評分系統
- **詞彙題（1-10）**：每題1分
- **綜合測驗（11-20）**：每題1分
- **文意選填（21-30）**：每題1分
- **篇章結構（31-34）**：每題2分
- **閱讀測驗（35-46）**：每題2分
- **混合題（49）**：多選題4分，採部分給分公式 `(2n-k)/n`

### 5. 響應式設計
- **桌機**：側邊導覽 + 主要作答區
- **平板**：可收合導覽列
- **手機**：堆疊式布局，觸控友善

## 🔧 技術規格

- **前端技術**：HTML5 + CSS3 + Vanilla JavaScript
- **資料儲存**：localStorage
- **相容性**：Chrome, Firefox, Safari, Edge
- **無需安裝**：直接開啟 HTML 檔案即可使用
- **離線可用**：不需網路連線

## 📊 評分標準

### 客觀題（62分）- 自動評分
- 詞彙題：10分
- 綜合測驗：10分
- 文意選填：10分
- 篇章結構：8分
- 閱讀測驗：24分

### 主觀題（38分）- 需人工評量
- 中譯英：8分（2題，各4分）
- 英文作文：20分

## 💡 使用建議

1. **模擬正式考試**：建議在安靜環境，完整作答100分鐘
2. **分段練習**：可針對特定題型反覆練習
3. **檢討錯題**：作答後務必查看 `review.html` 詳解
4. **記錄成績**：建議記錄每次測驗成績，追蹤進步情況

## 🐛 已知限制

- 翻譯與作文題需另外作答（系統僅提供參考答案）
- 多選題（第49題）需全對才給滿分
- 填充題（第47、48、50題）需完全符合答案才計分
